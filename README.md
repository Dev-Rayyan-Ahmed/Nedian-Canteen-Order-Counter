# Nedian-Canteen-Order-Counter
A CLI-based Canteen Ordering System using C Language.

## Description:
This project is a command-line interface (CLI) application for managing canteen orders. It was built during the Computer Engineering Workshop (CEW) course at NED University to facilitate a faster and easier ordering experience.
